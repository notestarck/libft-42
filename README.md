# Libft Ecole 42

Library of usual functions. First project of the Ecole 42.

## Languages

* C

## How to compile

In your terminal
```
libft-42 » make
```

## How to delete

In your terminal

To delete .o files
```
libft-42 » make clean
```

To delete libft.a
```
libft-42 » make fclean
```

## How to use it

Include the library in the makefile of your project
```
LIBS	=	-L./ -lft
```
